# Sambilan.id

Implementasi desain Figma untuk platform Sambilan.id menggunakan React, Vite, dan Tailwind CSS.

## Fitur yang Diimplementasikan
- **Halaman Pencarian**: Dilengkapi dengan search bar, saran pencarian, dan asisten AI.
- **Hasil Pencarian**: Menampilkan daftar quest dengan filter kategori dan urutan.
- **Detail Quest**: Informasi lengkap mengenai quest, termasuk reward, lokasi, dan profil pembuat.
- **Navigasi Mobile**: Bottom navigation bar dan top app bar yang sesuai dengan desain.

## Stack Teknologi
- **Frontend**: React (TypeScript)
- **Styling**: Tailwind CSS 4.x
- **Icons**: Lucide React
- **Build Tool**: Vite

## Cara Menjalankan
1. Instal dependensi: `npm install`
2. Jalankan mode pengembangan: `npm run dev`
3. Buka browser di `http://localhost:5173`

## Navigasi Demo
Ketuk di mana saja pada layar untuk berpindah antar halaman (Search -> Results -> Detail).
